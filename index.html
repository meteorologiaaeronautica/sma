Sistema de Meteorología Aeronáutica

Sistema de Meteorología Aeronáutica (SMA) es un software que recopila y analiza datos meteorológicos a nivel nacional. Transforma estos datos en valores fáciles de interpretar, facilitando así la obtención de resultados claros y útiles.

Modulos

Dashboard

Datos de todos los aerodromos donde se muestra los datos estadisticos, una vez ya procesados los datos

Login

Se debe ingresar con un usuario y contrasena, para identificarse y asi poder usar el sistema

Resumen

Muestra lso datos estadistios y totales, en un determinado rango de fecha y tambien en un determinado aerodromo y departamento

Historial

Se registran los datos procesados para un mejor control de ellos y asi poder conultarlos cuando sea necesario
